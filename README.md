### Requirements
    Python >= 3.x
    Flask
    PyMySQL

## Installation

1. Clone and navigate to Hall-Booking directory.

2. Run the python script.
    ```bash
    python3 __init__.py
    ```
3. You're all done :)
4. To quit ctrl+c.
