### Requirements
    Python >= 3.x

    PyMySQL

## Installation

1. Clone and navigate to Hall-Booking directory.

2. Run the Server (XAMPP/LAMPP).<br>
   for linux goto lampp directory and do-
    ```bash
    sudo ./xampp start
    ```
3. Run the python script.
    ```bash
    python3 __init__.py
    ```
4. You're all done :)
5. To quit ctrl+c.
